# Glypher Next.js example

Next.js App Router example using `@glypher/react`, `StarterKit` and the default theme.

## Run

```bash
pnpm install
pnpm --filter @glypher/example-nextjs dev
```

Open <http://localhost:3030>.

## What it demonstrates

- client-only editor usage with `"use client"`
- `StarterKit`
- a small custom toolbar
- saving HTML and JSON from `onUpdate`
